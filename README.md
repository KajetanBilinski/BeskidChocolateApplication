## About

Beskid Chocolate Factory is a renowned chocolate manufacturer nestled in the picturesque Beskid Mountains. With this app, you can embark on a virtual tour of the factory, which includes a step-by-step journey through the chocolate-making process. Learn about the history, ingredients, and craftsmanship that goes into each delectable chocolate creation.

The primary goal of this app is to provide a tantalizing glimpse of the factory's inner workings, enticing you to visit in person and savor the delightful chocolates firsthand. It's an opportunity to explore and appreciate the art of chocolate-making.

## Getting Started

To get started with the Beskid Chocolate Factory Tour App, follow these simple steps:

1. **Clone the Repository:**

```bash
git clone https://github.com/yourusername/beskid-chocolate-app.git
```

2. **Go to the main app folder:**

```bash
cd beskid-chocolate-app
```

3. **Install Dependencies:**

```bash
flutter pub get
```

4. **Run the App:**

```
flutter run
```
Start exploring the world of chocolate right from your device!

The app is also available on Google Play !
